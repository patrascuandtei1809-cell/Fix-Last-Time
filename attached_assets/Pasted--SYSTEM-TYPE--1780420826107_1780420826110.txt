  - compute indicators per module

  - compute weighted institutional score

STEP 9: CONFIDENCE ENGINE

  - apply deterministic rules
  - output: BUY / HOLD / SELL / NO TRADE

  - store in DB

STEP 12: AUDIT LOG

====================================================
DECISION ENGINE (NOW FULLY EXECUTABLE)
====================================================

INPUTS:

- confidence_score
- liquidity_risk
- institutional_score
- data_state

RULES:

IF data_state == UNKNOWN:
  RETURN "NO TRADE"

IF confidence < 60:
  RETURN "NO TRADE"

IF liquidity_risk > 70:
  RETURN "AVOID"

ELSE:

score mapping:

90-100 → STRONG BUY
75-89 → BUY
55-74 → HOLD
40-54 → REDUCE
20-39 → SELL
0-19 → AVOID
